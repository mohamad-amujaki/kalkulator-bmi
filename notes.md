# Catatan Belajar Membuat Kalkulator BMI dengan Python dan Streamlit

## Check Konfigurasi dan Buat Virtual Environment Baru

- python3 --version -> Check versi Python
- python3 -m venv .venv -> Buat virtual environment baru dengan nama .venv
- source .venv/bin/activate -> Aktifkan virtual environment baru

## Install Streamlit dan Check Library yang telah terinstall

- pip3 install streamlit
- pip3 freeze
